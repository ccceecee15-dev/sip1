Add a new **Reports → Exit Planning** screen to the existing application.

The UI must follow the **same layout, table components, filter design, spacing, and styling already used across the app** so it feels like part of the existing product.

The Exit Planning screen will use the **same dataset as Style Analysis**, but focused on identifying styles that may need **inventory exit or clearance planning**.

IMPORTANT: **All columns from the Excel dataset must be available in this screen**, even if some are hidden by default.

---

# Screen Header

Title: **Exit Planning**

Right side actions:

* Export to Excel
* Save View
* Reset Filters

Display summary cards above the table.

Cards:

# Exit Buckets

Add quick filter tabs to help planners identify potential exit items.

Buckets should be calculated using **Store WOS**:

Monitor → WOS 6–8
Slow Moving → WOS 8–12
Exit Candidate → WOS 12–20
Immediate Exit → WOS greater than 20

Selecting a bucket should filter the table.
