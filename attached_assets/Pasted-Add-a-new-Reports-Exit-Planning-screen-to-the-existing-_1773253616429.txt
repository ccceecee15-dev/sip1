Add a new **Reports → Exit Planning** screen to the existing application.

The UI must use the **same layout, typography, table component, filters, and styling already used across the application** so that the screen feels consistent with the existing reports module.

This screen will replicate the **Exit Planning Excel report**, which summarizes inventory aging by **MA (Merchandise Area)**.

The report must support **three table views** that match the Excel sheet:

1. Total Stock Aging
2. EOL Aging
3. Markdown Aging (MR1–MR4)

Users should be able to switch between these views using **tabs above the table**.

---

# Screen Header

Title: **Exit Planning**

Right side actions:

Export to Excel
Reset Filters

---

# Filters

Place a small filter panel above the report.

Filters should include:

Merchandise Area (MA)
Category
Sub Category
Brand

Buttons:

Apply Filters
Clear Filters

---

# View Tabs

Add three tabs above the table.

Total Stock Aging
EOL Aging
Markdown Aging

Switching tabs should update the table columns.

---
