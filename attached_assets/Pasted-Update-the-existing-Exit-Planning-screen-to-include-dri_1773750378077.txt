# Drill-Down Behavior

When a user clicks on any row in the Exit Planning table (MA level):

Open a **right-side drawer or expandable panel** showing detailed breakdown.

This panel must include two sections:

1. Markdown Details (Style-level)
2. Store Details (Store-level)
