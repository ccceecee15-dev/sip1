1. Conceptual model (important)

This screen now shows uplift as an index, not demand.

Baseline uplift = 1.0

Weekly uplift values vary above or below 1.0

e.g. 0.8, 1.0, 1.2

Values represent multipliers, not units

Do not show absolute demand numbers.
