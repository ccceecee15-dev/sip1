Facings Summary

Rows = Product Group (e.g. TRUE WRSS IN EAR, WIRED IN EAR)

On-shelf Capacity Summary

Same row/column structure

Values = net physical capacity over / under

Negative values visually flagged

This is a diagnostic view only

👉 These values must update immediately when SKU rows change.

Product Group (read-only)

SKU ID (read-only)

SKU Description (read-only)

SKU Grade (read-only; informational only)

No. of Facings ✅ editable

Single Capacity (read-only)

Allocation (read-only or auto-derived)

Total On-shelf (derived, read-only)

✅ No. of Facings

Summary matrices

Product Group

Store Grade
