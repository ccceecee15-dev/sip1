Purpose of the Screen

This screen defines physical on-shelf rules for SKUs:

How many facings a SKU gets

What minimum presentation quantity is required

These rules constrain SIP later but are not inventory planning.

Sanity Check

If a user asks:

“Is this how much we’ll stock?”

The correct answer should be:

“No — this defines how the product must appear on shelf.”
